## 快速开始

### 1. 创建第一部小说

1. 点击左侧边栏的 **"小说管理"**
2. 点击 **"+ 新建小说"**
3. 填写小说信息：
   - 标题
   - 简介
   - 类型（玄幻、都市、科幻等）
   - 目标字数
4. 点击 **"创建"**

### 2. 添加章节

**方法一：手动创建**
1. 进入小说编辑器
2. 点击 **"+ 新建章节"**
3. 输入章节标题
4. 开始写作

**方法二：导入 TXT**
1. 点击 **"导入 TXT"**
2. 选择文件
3. 系统自动识别章节（支持多种格式）

**方法三：从大纲生成**
1. 创建大纲结构
2. 选择大纲节点
3. 点击 **"生成章节"**

## AI 创作助手

### 三种 AI 角色

**💬 助手模式**
- 回答创作问题
- 提供写作建议
- 解答疑惑

**✍️ 作家模式**
- 改写内容
- 润色文字
- 扩写段落

**🔍 审校模式**
- 检查 OOC（人物性格不一致）
- 发现逻辑漏洞
- 提出改进建议

### 风格控制矩阵

**保留度（20-100%）**
- 20%：大幅改写
- 50%：适度改写
- 100%：保持原文

**扩写倾向**
- 保守：精炼简洁
- 适中：平衡详略
- 激进：详细描写

**内容尺度**
- 全年龄：适合所有读者
- 普通：一般成人向
- 成熟向：包含成人内容

**文风倾向**
- 自定义输入（如：网文轻小说、古典文学、硬科幻等）

### RAG 记忆系统

**功能：** AI 能记住全书内容，检索相关前文剧情

**使用方法：**
1. 系统自动索引章节内容
2. AI 对话时自动检索相关记忆
3. 提供更连贯的创作建议

**查看记忆统计：**
- 点击 **"RAG 记忆"** 按钮
- 查看已索引的章节数量

## 智能分析系统

### 综合分析

**功能：** 一站式分析，整合所有维度

**包含内容：**
- 综合评分（0-100分）
- 等级评定（S/A/B/C/D）
- 优先级问题识别
- 改进建议生成
- 详细综合报告

**使用方法：**
1. 打开章节编辑器
2. 点击 **"智能分析"** 标签
3. 选择 **"综合分析"**
4. 点击 **"开始分析"**

### 写作风格分析

**分析维度：**
- 五感描写（视觉、听觉、嗅觉、味觉、触觉）
- 对话质量（水对话、说明书对话、缺乏个性）
- 动作描写（平铺直叙、节奏单调）
- 场景渲染（缺乏细节、静态描写）
- 心理描写（直白心理、过度心理）

**使用方法：**
1. 选择 **"写作风格"** 标签
2. 点击 **"分析"**
3. 查看各维度评分和建议

### 情节张力分析

**分析维度：**
- 冲突强度（人物冲突、环境冲突、内心冲突）
- 悬念设置（疑问句、未解之谜、危机预告、伏笔）
- 转折点识别（反转、意外、揭秘）
- 高潮布局（强度、铺垫、解决完整度）
- 节奏控制（句式变化、段落密度、呼吸感）

### 情绪曲线追踪

**功能：** 追踪章节的情绪变化

**分析内容：**
- 情绪词汇识别（8种基本情绪）
- 情绪强度评分（-100到100）
- 情绪起伏曲线
- 情绪传递效果
- 情绪平衡度

### 网文能力分析

**分析内容：**
- 7种核心写作模式（黄金三章、打脸循环、力量进阶等）
- 6种爽点类型（力量、打脸、宝物、突破、认可、复仇）
- 黄金三章分析
- 爽点密度检测
- 章末钩子识别

### 内容检查

**检查项目：**
- 专有名词一致性（人名、地名、道具名）
- 敏感词检测（四级分类）
- AI味检测和评分
- 批量去AI味处理

## 大纲管理

### 创建大纲

**方法一：手动创建**
1. 点击 **"大纲管理"**
2. 点击 **"+ 添加节点"**
3. 选择节点类型（卷/章节/场景/备注）
4. 填写标题和内容

**方法二：使用模板**
1. 点击 **"应用模板"**
2. 选择模板：
   - 三幕式结构
   - 五幕式结构
   - 英雄之旅
   - 网文爽文结构
   - 玄幻修仙结构
   - 悬疑推理结构
3. 点击 **"应用"**
4. 根据需要修改

### 大纲操作

**拖拽排序**
- 拖动节点调整顺序
- 拖动到其他节点下成为子节点

**状态标记**
- 📋 计划中
- ✍️ 写作中
- ✅ 已完成

**进度追踪**
- 设置目标字数
- 查看实际字数
- 完成度百分比

### 章节关联

**功能：** 大纲节点和实际章节双向关联

**使用方法：**
1. 选择大纲节点
2. 点击 **"关联章节"**
3. 选择对应的章节
4. 系统自动同步状态

**自动匹配：**
- 系统根据标题自动匹配
- 手动确认或调整

### 一键生成章节

**功能：** 从大纲节点生成章节

**使用方法：**
1. 选择大纲节点
2. 点击 **"生成章节"**
3. 系统自动创建章节并填充内容：
   - 章节标题
   - 大纲内容作为注释
   - 目标字数提示
   - 占位符

**批量生成：**
1. 选择多个节点
2. 点击 **"批量生成"**
3. 系统依次生成所有章节

### 大纲导出

**支持格式：**
- Markdown
- HTML
- JSON
- 纯文本

**使用方法：**
1. 点击 **"导出大纲"**
2. 选择格式
3. 点击 **"下载"**

### 大纲统计

**统计内容：**
- 节点总数
- 按类型统计
- 按状态统计
- 目标字数 vs 实际字数
- 完成度
- 写作速度
- 预计完成时间

## 常见问题

### Q1: 如何导入已有的小说？

**A:** 使用 TXT 导入功能：
1. 准备好 TXT 文件
2. 确保章节标题格式正确（如：第一章 标题）
3. 点击 **"导入 TXT"**
4. 选择文件
5. 系统自动识别章节

**支持的章节格式：**
- `第X章 标题`
- `第X回 标题`
- `Chapter X 标题`
- `X. 标题`

### Q2: AI 改写后如何对比？

**A:** 使用 Diff 对比编辑器：
1. AI 改写完成后自动显示对比界面
2. 左侧显示原文，右侧显示改写后的文本
3. 差异部分高亮显示
4. 点击 **"应用"** 采用改写结果
5. 点击 **"取消"** 保留原文

### Q3: 如何备份我的小说？

**方法一：数据备份功能**
1. 点击 **"设置"**
2. 点击 **"数据备份"**
3. 点击 **"导出备份"**
4. 保存 JSON 文件

**方法二：导出功能**
1. 选择小说
2. 点击 **"导出"**
3. 选择格式（TXT/Markdown/Word/PDF）
4. 下载文件

### Q4: RAG 记忆系统如何工作？

**A:** RAG（检索增强生成）系统：
1. 系统自动索引所有章节内容
2. 使用 TF-IDF + 余弦相似度算法
3. AI 对话时自动检索相关内容
4. 提供更准确的创作建议

**注意：**
- 首次使用需要索引，可能需要几秒钟
- 修改章节后会自动重新索引
- 可以手动清除记忆重新索引

### Q5: 如何提高 AI 改写质量？

**建议：**
1. **调整保留度**：根据需要选择合适的保留度
2. **明确文风**：在文风倾向中输入具体的风格描述
3. **提供上下文**：确保 RAG 记忆系统已索引相关章节
4. **多次尝试**：使用多方案续写，选择最佳结果
5. **分段处理**：对于长文本，分段处理效果更好

### Q6: 智能分析结果如何使用？

**建议：**
1. **查看综合评分**：了解整体质量
2. **关注优先级问题**：先解决影响最大的问题
3. **应用改进建议**：按照建议逐步优化
4. **使用优化提示词**：直接用于 AI 改写
5. **定期分析**：每写完一章进行分析

### Q7: 如何设置写作目标？

**A:**
1. 点击 **"写作统计"**
2. 点击 **"设置目标"**
3. 选择目标类型：
   - 每日目标
   - 每周目标
   - 每月目标
   - 总字数目标
4. 输入目标字数
5. 点击 **"保存"**

### Q8: 专注模式下如何操作？

**A:** 专注模式（F11）下：
- 只显示编辑器
- 隐藏所有工具栏和侧边栏
- 按 `Esc` 或 `F11` 退出
- 自动保存仍然工作
- 快捷键仍然可用

### Q9: 如何自定义快捷键？

**A:**
1. 点击 **"设置"**
2. 点击 **"快捷键设置"**
3. 找到要修改的功能
4. 点击输入框
5. 按下新的快捷键组合
6. 系统自动检测冲突
7. 点击 **"保存"**

### Q10: 数据存储在哪里？

**A:** 
- 所有数据存储在浏览器的 localStorage 中
- 数据仅保存在本地，不会上传到服务器
- 建议定期使用数据备份功能导出备份
- 清除浏览器数据会删除所有小说数据
