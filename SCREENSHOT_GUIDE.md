## 🎬 完整体验流程

### 1. 启动项目
```bash
cd /home/ubuntu/QDtiandaoxiaoshuo
npm run dev
```

### 2. 访问应用
打开浏览器访问：http://localhost:3001

### 3. 体验专注模式
- 点击左侧"创作工具"
- 点击工具栏的 📐 图标或按 F11
- 体验全屏写作
- 按 ESC 退出

### 4. 体验夜间模式
- 点击顶部导航栏右侧的 🌙 图标
- 观察所有页面切换为深色主题
- 刷新页面，验证主题保持
- 再次点击 ☀️ 切换回浅色

### 5. 体验全文搜索
- 按 Ctrl+K 打开搜索
- 输入关键词（如"小说"）
- 观察实时搜索结果
- 点击一个结果，跳转到对应内容
- 按 ESC 关闭搜索
