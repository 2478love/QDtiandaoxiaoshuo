# 个性化功能演示指南

## 🎯 功能概览

本次更新为天道小说创作系统添加了三个核心个性化功能：

1. **📝 模板管理** - 快速创建小说的模板系统
2. **🏷️ 标签系统** - 小说分类和筛选
3. **📊 写作统计** - 可视化写作数据

---

## 📝 功能 1：模板管理

### 访问路径
1. 登录系统
2. 点击侧边栏"小说管理"
3. 在小说管理页面，可以看到模板功能（需要集成到 NovelManager）

### 功能特性
✅ **6个预设模板**
- 玄幻小说模板 - 经典修仙、异界题材
- 都市小说模板 - 现代都市题材
- 科幻小说模板 - 未来世界、星际探索
- 武侠小说模板 - 江湖恩怨、侠义故事
- 战斗章节模板 - 战斗场景结构
- 日常章节模板 - 日常生活场景

✅ **自定义模板**
- 创建个人模板
- 编辑模板内容
- 删除自定义模板（预设模板不可删除）

✅ **模板应用**
- 一键应用模板到新小说
- 自动填充标题、简介、大纲

### 使用示例
```typescript
// 创建模板
const template = {
  name: "我的玄幻模板",
  description: "适合修仙题材",
  category: "novel",
  content: {
    outline: "第一章：开局\n第二章：奇遇\n..."
  }
};

// 应用模板
TemplateService.applyTemplate(template);
```

## 🏷️ 功能 2：标签系统

### 访问路径
1. 登录系统
2. 在"小说管理"页面顶部可以看到标签筛选器
3. 创建/编辑小说时可以添加标签

### 功能特性
✅ **10个预设标签**
- 玄幻 (紫色 #9333ea)
- 都市 (蓝色 #3b82f6)
- 科幻 (青色 #06b6d4)
- 武侠 (橙色 #f59e0b)
- 仙侠 (紫罗兰 #8b5cf6)
- 历史 (琥珀色 #d97706)
- 悬疑 (深紫色 #7c3aed)
- 完结 (绿色 #10b981)
- 连载 (红色 #ef4444)
- 草稿 (灰色 #6b7280)

✅ **自定义标签**
- 创建个性化标签
- 选择标签颜色（10种预设 + 自定义）
- 删除自定义标签（预设标签不可删除）

✅ **标签筛选**
- 按标签筛选小说
- 显示每个标签的使用次数
- 支持多标签组合

### 使用示例
```typescript
// 添加标签
TagService.addTag("热血", "#ef4444");

// 筛选小说
const filtered = TagService.filterNovelsByTag(novels, "tag_xuanhuan");

// 更新标签计数
const tags = TagService.updateTagCounts(novels);
```

## 📊 功能 3：写作统计

### 访问路径
1. 登录系统
2. 进入"仪表盘"
3. 滚动到页面底部查看"写作统计"区块

### 功能特性
✅ **核心统计**
- 总字数 - 所有作品累计字数
- 今日字数 - 当天写作字数
- 本周字数 - 本周累计字数
- 作品数量 - 总数、完结、连载

✅ **写作趋势**
- 最近7天写作字数柱状图
- 每日写作时长统计
- 趋势可视化展示

✅ **目标管理**
- 设置每日写作目标（默认 2000 字）
- 设置每周写作目标（默认 10000 字）
- 实时显示目标完成进度
- 完成目标时显示祝贺提示

✅ **详细统计**
- 总作品数、总章节数
- 完结作品、连载作品
- 平均每部字数、平均每章字数

### 使用示例
```typescript
// 获取统计数据
const stats = StatsService.getStats(novels);

// 记录写作活动
StatsService.recordWriting(1500, 30); // 1500字，30分钟

// 设置目标
StatsService.setGoals(2000, 10000);

// 获取写作趋势
const trend = StatsService.getWritingTrend(7);
```

## 🚀 快速开始

### 1. 使用模板创建小说
```bash
1. 进入"小说管理"
2. 点击"模板"标签
3. 选择"玄幻小说模板"
4. 点击"使用"
5. 填写小说标题
6. 添加标签（玄幻、连载）
7. 开始创作
```

### 2. 管理标签
```bash
1. 进入"小说管理"
2. 点击"+ 添加标签"
3. 输入标签名称（如"热血"）
4. 选择颜色
5. 点击"添加"
6. 在小说列表使用标签筛选
```

### 3. 查看统计
```bash
1. 进入"仪表盘"
2. 滚动到"写作统计"区块
3. 查看今日字数和趋势
4. 点击"设置目标"
5. 设置每日/每周目标
6. 开始写作，系统自动记录
```
